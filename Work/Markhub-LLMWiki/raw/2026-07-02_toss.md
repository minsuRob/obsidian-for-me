# 토스페이 결제 연동 회의 (AI 요약)

- 전반적으로 스펙은 합의됐는데 일정이 밀림.

## 나온 이야기들 (섞임 주의)

- 원래 결제 연동 오픈 목표가 **7월 말**이었는데, 토스 쪽 심사 일정 때문에 **8월 중순**으로 연기하기로 함.
- 수수료 협상: 기존 협의안이 2.0% 였는데, 이번에 볼륨 기준 걸고 **1.8%로 인하** 합의. (월 거래액 5천만원 이상 조건)
- 개발 스펙:
    - 결제창은 SDK 방식 대신 **리다이렉트 방식**으로 가기로. (Electron 웹뷰 이슈 때문)
    - 웹훅으로 결제 상태 콜백 받는 구조. 멱등키 필수.
    - 정산 API는 2차 스코프로 미룸.
- 리스크: 토스 심사 기간이 더 길어지면 8월 중순도 위험할 수 있음. 버퍼 1주 잡아두기로.
- 액션아이템:
    - 나: 리다이렉트 결제 플로우 스펙 문서화 (금주 내)
    - 김PM: 수수료 1.8% 계약서 반영 요청
