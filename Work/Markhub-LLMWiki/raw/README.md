# raw/ — 원본을 던지는 곳

여기는 **아무렇게나 던져도 되는 입구**입니다. 형식 걱정하지 마세요.

## 무엇을 넣나
- 오늘 있었던 일 메모
- 회의록 (녹취/받아쓰기 그대로 붙여넣기 OK)
- Slack 대화 복붙, 이메일 복붙
- 아이디어, 스크린샷, 링크

## 어떻게 쓰나
1. **오늘 기록** → `daily/` 안에 날짜별 노트(`2026-07-02.md`)에 적습니다.
   - 데일리노트 단축키(옵시디언)로 자동 생성됩니다. 안 되면 그냥 파일 하나 만들어도 됩니다.
2. **그 외 아무 메모/회의록** → 이 `raw/` 폴더에 파일로 던집니다.

## 규칙 딱 하나
- **여기 넣은 원본은 AI가 절대 고치지 않습니다.** 원본은 그대로 남고, 정제된 결과는 `../wiki/`에 쌓입니다.
