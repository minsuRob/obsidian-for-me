# Feature: 토스페이 결제 연동

## 개요
- 앱 내 결제를 토스페이로 연동하는 기능.
- 사용자가 상품 결제 시 토스페이 결제창을 통해 결제하고, 웹훅으로 상태를 콜백받는다.

## 진행 상황
- **오픈 목표: 8월 중순** (토스 심사 일정으로 7월 말 → 연기, 버퍼 1주)
- 결제 방식: **리다이렉트 방식** (Electron 웹뷰 이슈로 SDK 대신)
- 수수료율: **1.8% 합의** (월 거래액 5천만원 이상 조건) → [[Vendor-TossPay]]
- 콜백: 웹훅으로 결제 상태 수신, 멱등키 필수
- 정산 API: 2차 스코프로 이연

## 결정 사항
- 결제창은 리다이렉트 방식 채택 ([[Feature-Electron배포]] 웹뷰 이슈 대응)
- 웹훅 + 멱등키 구조로 결제 상태 콜백
- 정산 API는 1차 범위에서 제외

## 리스크
- 토스 심사 기간이 더 길어지면 8월 중순도 위험 (버퍼 1주 확보)

## 관련 링크

- 관련 이슈: [[Feature-Electron배포]] (웹뷰 결제창 이슈)
